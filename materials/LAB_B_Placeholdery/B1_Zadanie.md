# LAB B: Placeholdery a opakovateľné šablóny

## Cieľ

Z jednorazového promptu vytvoríme šablónu, ktorú vieme použiť v rôznych témach bez opakovaného prepisovania stabilných pravidiel.

## Východiskové zadanie

```text
Navrhni mi päť tém pre kvalitné SEO blogové príspevky o Exceli pre účtovníkov.
```

## Úloha 1: Identifikácia premenných

Nahraďte konkrétne hodnoty placeholdermi:

- počet tém,
- typ obsahu,
- oblasť,
- cieľová skupina,
- komunikačný cieľ,
- formát výstupu,
- tón,
- obmedzenia.

## Úloha 2: Vytvorenie šablóny

Použite hranaté zátvorky:

```text
Navrhni [počet] tém pre [typ obsahu] v oblasti [oblasť].

Cieľová skupina:
[cieľová skupina]

Cieľ:
[cieľ]

Pri každej téme uveď:
[požadované položky]

Pravidlá:
[stabilné pravidlá]

Výstup:
[formát]
```

## Úloha 3: Dva testy

Šablónu spustite dvakrát.

### Test A

- oblasť: dátová analytika,
- cieľová skupina: manažéri,
- cieľ: vysvetliť praktické využitie dashboardov.

### Test B

- oblasť: kybernetická bezpečnosť,
- cieľová skupina: malé firmy,
- cieľ: zvýšiť povedomie o phishingu.

## Úloha 4: Kontrola šablóny

| **Otázka** | **Áno/Nie** | **Potrebná oprava** |
|---|---:|---|
| Je význam každého placeholdera jasný? | | |
| Funguje šablóna v oboch témach? | | |
| Zostáva štruktúra výstupu rovnaká? | | |
| Neobsahuje šablóna jednorazové údaje? | | |
| Sú pravidlá dostatočne konkrétne? | | |
| Je možné výsledky medzi sebou porovnať? | | |

## Úloha 5: Vlastná pracovná šablóna

Vyberte jednu oblasť:

- pracovný e-mail,
- blogový článok,
- príspevok na sociálnu sieť,
- analýza námietok,
- plán aktivity,
- spätná väzba k dokumentu.

Vytvorte vlastnú šablónu s minimálne šiestimi placeholdermi a tromi stabilnými pravidlami.

## Kritériá úspechu

- každý placeholder má jednoznačný názov,
- stabilné pravidlá nie sú ukryté v meniteľných hodnotách,
- šablóna funguje v dvoch odlišných situáciách,
- výsledok má konzistentnú štruktúru,
- šablóna nevyžaduje doplňujúce vysvetlenie pri každom použití.
