# LAB A: Ukážkové prompty

## 1. Slabý prompt

```text
Napíš mi blog o AI.
```

Problémy:

- nie je určená téma,
- nepoznáme cieľovú skupinu,
- nepoznáme účel,
- chýba rozsah a štruktúra,
- chýbajú zdroje a obmedzenia,
- nevieme posúdiť kvalitu výsledku.

## 2. Použiteľný prompt

```text
Napíš blogový článok o praktickom používaní ChatGPT pre začiatočníkov.
Použi jednoduchý jazyk, krátke odseky a päť praktických tipov.
Rozsah maximálne 900 slov.
```

Zlepšenie: téma, publikum, štýl a rozsah sú určené. Stále chýba presný cieľ, vstupy, štruktúra a kontrola.

## 3. Kvalitný prompt

```text
Napíš vzdelávací blogový článok na tému „Ako vytvoriť kvalitný prompt pre ChatGPT“.

Cieľová skupina:
Začiatočníci, ktorí používajú ChatGPT na pracovné e-maily, obsah a plánovanie.

Cieľ:
Po prečítaní má čitateľ vedieť vytvoriť prompt s cieľom, kontextom, vstupmi, pravidlami, formátom a kontrolou.

Obsahové vstupy:
- vysvetli rozdiel medzi témou a cieľom,
- ukáž slabý a kvalitný prompt,
- vysvetli použitie placeholderov,
- pridaj kontrolný zoznam.

Štruktúra:
- H1,
- krátky úvod,
- šesť častí podľa skladby promptu,
- praktický príklad,
- kontrolný zoznam,
- záver.

Pravidlá:
- jednoduchý, vecný a profesionálny jazyk,
- maximálne 1 100 slov,
- bez vymyslených štatistík,
- bez tvrdenia, že kvalitný prompt zaručuje správnu odpoveď,
- pri časovo citlivých funkciách upozorni na potrebu overenia.
```

## 4. Prompt na prepis textu pre inú cieľovú skupinu

```text
Prepracuj nasledujúci text pre [cieľová skupina].

Pôvodný text:
[text]

Cieľ:
[čo má čitateľ pochopiť alebo urobiť]

Tón:
[tón]

Úroveň odbornosti:
[začiatočník/mierne pokročilý/expert]

Zachovaj:
- všetky overené fakty,
- názvy produktov a dokumentov,
- dôležité upozornenia.

Zmeň:
- terminológiu podľa publika,
- dĺžku viet,
- poradie informácií podľa praktickej dôležitosti.

Nevymýšľaj nové fakty.
```

## 5. Prompt na kritiku vlastného výstupu

```text
Skontroluj predchádzajúci výstup ako prísny editor.

Hodnoť:
- splnenie cieľa,
- vhodnosť pre cieľovú skupinu,
- presnosť,
- konkrétnosť,
- štruktúru,
- zbytočné frázy,
- možné vymyslené alebo nepodložené tvrdenia.

Výstup:
1. tri najsilnejšie časti,
2. päť problémov podľa závažnosti,
3. konkrétne opravy promptu,
4. prepracovaná verzia iba po mojom potvrdení.
```

## 6. Prompt na označenie predpokladov

```text
Pred vytvorením odpovede skontroluj, či máš všetky potrebné vstupy.

Ak niečo chýba:
- polož najviac tri nevyhnutné otázky,
- alebo uveď explicitný PREDPOKLAD,
- nevydávaj predpoklad za fakt.

Potom vytvor výsledok podľa zadania.
```

## Praktická zásada

Prompt zlepšujeme konkrétnymi kritériami. Nepoužívame neurčité formulácie ako:

- „sprav to lepšie“,
- „buď kreatívnejší“,
- „napíš to profesionálne“,
- „daj tam viac hodnoty“.

Namiesto toho určujeme pozorovateľnú zmenu:

- skráť o 30 %, 
- pridaj tri konkrétne kroky,
- odstráň marketingové frázy,
- nahraď odborné pojmy jednoduchými vysvetleniami,
- uveď jedno riziko pri každom odporúčaní.
